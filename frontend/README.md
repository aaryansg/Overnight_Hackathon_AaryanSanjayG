# Intelligent Document Processing Frontend

A React + Tailwind CSS application for uploading and processing infrastructure documents with AI-powered summarization and insights.

## Features
- Drag & drop document upload
- AI-powered document processing (simulated)
- Real-time processing status
- Document library with search
- Gemini AI insights display
- Responsive design

## Setup
1. Clone repository
2. Run `npm install`
3. Run `npm start`

## Tech Stack
- React 18
- Tailwind CSS
- Lucide React Icons
